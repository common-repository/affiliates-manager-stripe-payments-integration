Process an affiliate commission via Affiliates Manager after a Stripe Payments checkout

== Description ==

This addon allows you to integrate Stripe Payments with the Affiliates Manager plugin.

When a user pays for an item on your site, this addon will check to see if the user was referred to by an affiliate. It will then give commission to the affiliate who referred the user.

This addon requires the [Affiliates Manager](https://wordpress.org/plugins/affiliates-manager/) and [Stripe Payments](https://wordpress.org/plugins/stripe-payments/) plugins.

Read the usage instructions [here](https://wpaffiliatemanager.com/affiliates-manager-stripe-payments-integration/)

== Installation ==

1. Upload `affiliatemgr-stripe-payments-integration` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
